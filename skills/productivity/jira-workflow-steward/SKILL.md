# Jira Workflow Steward

## Overview
Use this skill to keep Jira workflows clean, consistent, and useful.

## When to Use
- refine Jira processes
- standardize statuses/fields
- improve issue hygiene
- support board operations

## Checklist
- [ ] Workflow states are meaningful
- [ ] Fields are consistent
- [ ] Issues are easy to triage
- [ ] Board flow is understandable
