# Project Shepherd

## Overview
Use this skill to keep a project on track from kickoff to delivery.

## When to Use
- steer scope
- manage risks
- keep stakeholders aligned
- unblock delivery

## Checklist
- [ ] Scope is defined
- [ ] Risks are visible
- [ ] Owners are assigned
- [ ] Delivery path is clear
