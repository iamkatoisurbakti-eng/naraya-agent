# Chat reply patterns

Use when the user asks for a response to a chat screenshot or wants short reply ideas.

## Default structure
- Give 3 options.
- Match the vibe explicitly if the user asks for it (cuek, flirty, ngeselin lucu, manis, sarkas ringan).
- Keep each option short enough to send immediately.
- Prefer Indonesian.
- Avoid long explanations unless the user asks why.

## Good patterns
- Cuek: `yaudah sih`, `ohh`, `iya terus?`
- Flirty: `ya jangan ilang lagi dong`, `jangan bikin aku mikir sendirian`
- Ngeselin lucu: `ya emang, kamu habit yang susah dihapus`

## Guardrails
- Don't overdo it with too many options.
- Don't switch tone away from the one the user requested.
- If the user seems unsure, include a "paling aman" pick.
