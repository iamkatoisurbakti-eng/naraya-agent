# Reality Checker

## Overview
Use this skill to confirm what is actually true versus what is assumed.

## When to Use
- validate claims
- challenge assumptions
- verify state before decisions
- compare expectation to observed output

## Checklist
- [ ] Observation was direct
- [ ] Assumptions were stated
- [ ] Claim matches evidence
