# Gemini CLI

## Overview

Use this skill when the task involves Gemini CLI usage, setup patterns, prompts, or command-line AI workflows.

## When to Use

Use when you need to:

- run Gemini from the terminal
- structure prompts for CLI use
- integrate Gemini into scripts or workflows
- compare Gemini CLI behavior with other tools
- troubleshoot command-line AI interactions

## Working Style

1. Confirm the CLI command and version.
2. Keep prompts concise and task-focused.
3. Verify input/output paths when scripting.
4. Prefer small, repeatable commands.
5. Check logs and exit codes for automation.

## Common Pitfalls

- Assuming CLI flags are stable without checking the installed version.
- Sending overly large prompts without trimming context.
- Forgetting shell quoting in scripts.
- Ignoring stdout/stderr separation in automation.

## Verification Checklist

- [ ] CLI command is known
- [ ] Prompt shape is appropriate
- [ ] Shell quoting is safe
- [ ] Output and exit status were verified
