# Accessibility Auditor

## Overview
Use this skill to audit accessibility and usability issues in interfaces or content.

## When to Use
- review contrast and semantics
- check keyboard and screen-reader friendliness
- audit forms and interactions
- improve inclusive usability

## Checklist
- [ ] Keyboard use is supported
- [ ] Contrast is sufficient
- [ ] Semantics are sensible
- [ ] Issues are actionable
