# Antigravity

## Overview

Use this skill when work feels heavy, blocked, or overcomplicated and the goal is to remove friction.
It focuses on simplification, momentum, and low-drag execution.

## When to Use

Use when you need to:

- reduce process overhead
- simplify a complex workflow
- remove blockers and bottlenecks
- keep a task moving with minimal ceremony
- turn a messy problem into a clear next step

## Working Style

1. Identify the heaviest source of friction.
2. Strip the task down to the next smallest useful action.
3. Remove unnecessary steps, tools, or dependencies.
4. Verify progress quickly.
5. Repeat until momentum is restored.

## Common Patterns

- simplify before optimizing
- eliminate duplicate effort
- use the shortest path to signal
- keep only the essentials visible

## Common Pitfalls

- adding process to solve process overhead.
- optimizing before the shape of the work is clear.
- over-designing the escape from complexity.

## Verification Checklist

- [ ] The main friction source is identified
- [ ] The next step is simpler than before
- [ ] Unnecessary work was removed
- [ ] Progress is visible again
