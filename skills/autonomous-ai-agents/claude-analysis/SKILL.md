# Claude Analysis

## Overview

Use this skill when you need to analyze Claude-generated output, prompts, or agent workflows.
It helps identify strengths, weaknesses, missing steps, and opportunities for improvement.

## When to Use

Use when you need to:

- review Claude responses for quality
- inspect prompts or workflows for gaps
- evaluate whether output matches the goal
- suggest improvements to structure or behavior

Do not use for:

- direct code execution
- writing new features from scratch
- memory consolidation or retrieval

## Analysis Checklist

- Is the output on-task?
- Is the structure clear?
- Are there missing assumptions?
- Are there unsupported claims?
- Does the workflow need more verification?

## Common Pitfalls

1. Reviewing style without checking correctness.
2. Missing hidden assumptions in the prompt.
3. Failing to separate quality issues from factual issues.
4. Giving feedback without actionable next steps.

## Verification Checklist

- [ ] Goal alignment checked
- [ ] Correctness checked
- [ ] Structure checked
- [ ] Actionable improvements listed
