# Superpowers

## Overview

Use this skill as a general-purpose high-agency operating mode for complex work.
It emphasizes clear goals, disciplined execution, and explicit verification.

## When to Use

Use when you need to:

- tackle a complex task end-to-end
- combine planning, execution, and verification
- keep work structured across multiple steps
- communicate progress clearly and concisely
- avoid drifting into unverified assumptions

## Working Style

1. Clarify the goal and constraints.
2. Break the work into small, testable steps.
3. Execute the smallest safe step first.
4. Verify results before moving on.
5. Summarize outcomes and next actions plainly.

## Core Practices

- Prefer evidence over assumption.
- Keep the next action obvious.
- Verify each meaningful change.
- Reduce wasted motion.
- Keep communication direct and useful.

## Common Pitfalls

- Starting without a clear target.
- Doing too much before verifying anything.
- Losing track of intermediate results.
- Overexplaining instead of acting.

## Verification Checklist

- [ ] Goal is clear
- [ ] Steps are small and ordered
- [ ] Each meaningful step is verified
- [ ] Output is concise and actionable
