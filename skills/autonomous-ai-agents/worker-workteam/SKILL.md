# Worker Workteam

## Overview

Use this skill when a worker team needs a simple shared goal, clear handoffs, and a final completion check.
It is a lightweight coordination skill for team-style work.

## When to Use

Use when you need to:

- assign work across a small team
- define handoffs between workers
- keep a shared goal visible
- check completion before delivery

## Notes

- Make the shared goal explicit.
- Keep each handoff brief.
- End with one completion check.

## Checklist

- [ ] Shared goal set
- [ ] Handoffs clear
- [ ] Team output merged
- [ ] Completion verified
