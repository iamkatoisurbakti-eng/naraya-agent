# Sales Outreach

## Overview

Use this skill when writing or improving outreach that starts sales conversations.

## When to Use

- draft outbound messages
- build sequences
- improve reply rates
- tailor messages to persona and trigger
- test outreach hooks and CTAs

## Working Style

1. Identify the persona and trigger.
2. Keep the message short and relevant.
3. Lead with a clear reason to care.
4. Include one simple call to action.
5. Review response quality and iterate.

## Verification Checklist

- [ ] Persona is clear
- [ ] Message is relevant
- [ ] CTA is simple
- [ ] Sequence is coherent
- [ ] Test plan exists
