# 24-hour safe promotion loop

Use when the user asks for continuous/looping promotion, backtesting, debate/voting, or multi-agent marketing for 24 hours.

## Safe default
- Create strategy/evaluation loops, not unsolicited promotion.
- Do not send emails, post to social channels, scrape private contacts, or spam communities unless the user provides explicit channel access, recipient consent, final approved copy, and scope.
- Produce drafts, SEO changes, content calendars, outreach lists from public sources, and scored experiments.

## Loop pattern
1. Run an initial multi-perspective debate:
   - SEO/technical/content
   - Organic social/community
   - Conversion/email/retention/partnership
2. Score proposals by impact, speed, cost, risk, and product fit.
3. Schedule a 24-hour loop as 12 iterations every 2 hours.
4. Each iteration should:
   - review prior hypotheses/results if available
   - inspect public site/SEO surface if tools are available
   - refine keyword/content/channel priorities
   - draft 1-3 assets or experiments
   - vote on keep/kill/iterate
   - avoid irreversible external actions unless separately approved
5. Deliver results back to the originating chat/session when the platform supports it.

## Output format
- Timestamp/iteration number
- What changed since previous iteration
- Top 3 opportunities
- Draft copy/content assets
- Vote table
- Next action requiring user approval, if any
