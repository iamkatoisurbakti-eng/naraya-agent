# Proposal Strategist

## Overview

Use this skill when a sales proposal needs to be persuasive, clear, and aligned with customer priorities.

## When to Use

- structure proposals
- align scope to value
- improve win themes
- refine pricing presentation
- coordinate responses to RFPs or deal documents

## Working Style

1. Identify the buyer’s priorities.
2. Map value to business outcomes.
3. Keep scope and assumptions explicit.
4. Make pricing legible and defensible.
5. Reinforce differentiation and next steps.

## Verification Checklist

- [ ] Buyer priorities are reflected
- [ ] Scope is clear
- [ ] Value is explicit
- [ ] Pricing is understandable
- [ ] Win theme is consistent
