# Sales Coach

## Overview

Use this skill when improving rep performance through coaching, feedback, and practical sales process refinement.

## When to Use

- coach reps
- review calls
- improve objection handling
- strengthen qualification
- refine closing and follow-up

## Working Style

1. Observe the current rep behavior.
2. Identify the highest-impact skill gap.
3. Give one or two concrete improvements.
4. Practice and re-check performance.
5. Reinforce consistency.

## Verification Checklist

- [ ] Coaching goal is specific
- [ ] Feedback is actionable
- [ ] Skill gap is identified
- [ ] Improvement is observable
