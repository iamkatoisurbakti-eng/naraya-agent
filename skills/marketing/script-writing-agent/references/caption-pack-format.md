# Caption Pack Format

Session note: when the user asks for multiple news items with `caption + hook + CTA + 5 hashtags`, return a *direct publish-ready list*.

## Required shape per item
- `Hook/caption:` one short sentence that feels like a hook.
- `CTA:` one short sentence pointing to **News Nusantara AI**.
- `Hashtag:` exactly 5 hashtags.

## Output rules
- Use Indonesian.
- Keep it concise and clean.
- Do not add analysis, explanation, or tables unless the user asks.
- Do not exceed the requested hashtag count.
- Keep factual claims aligned to the source context; do not invent details.

## Example
1. **Hook/caption:** Data yang jujur bikin arah kebijakan makin tepat.
   **CTA:** Baca ringkasan lengkapnya di News Nusantara AI.
   **Hashtag:** #NewsNusantaraAI #NusantaraAINews #BeritaHariIni #EkonomiIndonesia #Banten
