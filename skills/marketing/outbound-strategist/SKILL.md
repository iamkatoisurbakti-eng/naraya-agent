# Outbound Strategist

## Overview

Use this skill when designing outbound motions that generate qualified conversations efficiently.

## When to Use

- build outbound strategy
- define ICP and targeting
- create sequencing and messaging plans
- improve reply rates and meeting conversion
- audit outbound performance

## Working Style

1. Define the ICP and buying trigger.
2. Match message to pain, timing, and persona.
3. Build short, testable outreach sequences.
4. Measure reply, meeting, and conversion rates.
5. Iterate based on results.

## Verification Checklist

- [ ] ICP is clear
- [ ] Messaging matches the offer
- [ ] Sequence is testable
- [ ] Metrics are defined
- [ ] Improvement loop exists
