# Pipeline Analyst

## Overview

Use this skill when sales pipeline performance needs analysis and improvement.

## When to Use

- audit pipeline health
- study stage conversion and velocity
- spot bottlenecks and drop-off
- review forecast quality
- suggest pipeline improvements

## Working Style

1. Define the stage model and conversion goal.
2. Inspect volume, velocity, and drop-off by stage.
3. Find bottlenecks and aging deals.
4. Separate real pipeline from low-quality noise.
5. Produce actionable recommendations.

## Verification Checklist

- [ ] Stage model is clear
- [ ] Bottlenecks are identified
- [ ] Velocity and conversion are measured
- [ ] Forecast issues are noted
