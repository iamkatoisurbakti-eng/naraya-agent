# Seedance 5x15 no-text prompts

Use this pack when the user asks for a *YouTube Shorts* style video built from 5 scenes x 15 seconds, with a newsy/breaking-news look and strict no-text output.

## Core requirements
- Vertical 9:16
- 5 scenes x 15 seconds each
- Realistic news / documentary b-roll
- No text anywhere in the frame or overlays
- Avoid logo, watermark, signage, menu text, subtitle cards, lower thirds
- Prefer clean hands, natural motion, shallow depth of field, handheld realism

## Recommended scene formula
1. Establishing shot: modern environment, newsy atmosphere, human activity
2. Action close-up: the main movement or subject event
3. Human mid shot: operator / subject / staff motion
4. Wider context shot: environment or surrounding activity
5. Hero closing shot: finished subject, calm cinematic ending
