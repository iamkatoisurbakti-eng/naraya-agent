# Session note: Shorts template from example

When the user shares an example link or says "template video seperti itu", treat it as a request for an extractable Shorts format.

## Practical handling
- If the example cannot be viewed directly, ask for a screenshot or a short description of the visual style.
- Default to a YouTube Shorts output spec in `9:16`.
- Produce a template that includes:
  - hook (0–3s)
  - main beats / body (3–20s)
  - closing CTA (final 2–5s)
- If the request is about news content, keep the tone concise, vertical, and caption-friendly.
- Prefer delivering the result as:
  - script template
  - scene-by-scene visual template
  - subtitle/caption layout
  - edit/export settings for Shorts

## Quality cues
- Keep the template easy to reuse across multiple clips.
- Make subtitle safe areas and title placement explicit for vertical video.
- Preserve the user's preferred short-form output shape rather than over-explaining the source example.
