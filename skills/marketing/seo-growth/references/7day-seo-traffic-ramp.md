# 7-Day SEO Traffic Ramp

Use this as a compact execution loop for a new SEO push on a live site.

## Day 1: Foundation
- Pick 1 primary keyword and 3–5 long-tail variants.
- Fix title, meta description, H1, canonical, robots, sitemap.
- Confirm indexability and no accidental noindex.

## Day 2: Pillar page
- Publish the main commercial-intent page.
- Add clear value prop, benefits, FAQ, CTA, and 3–5 internal links.

## Day 3: Long-tail support pages
- Publish 2 support pages targeting "hari ini" / "24 jam" / "terbaru" style intent.
- Keep each page focused on one search intent.

## Day 4: Trust / comparison page
- Publish 1–2 pages for "terpercaya", "alternatif", or comparison intent.
- Add proof points, FAQs, and clear next-step CTA.

## Day 5: Internal linking
- Link pillar <-> support pages.
- Use natural anchors; avoid repeating the exact same keyword every time.

## Day 6: CTR and snippet tuning
- Tighten title/meta for clickability.
- Add FAQ schema if appropriate.
- Ensure mobile layout is clean and fast.

## Day 7: Measure and decide
- Review Search Console impressions, clicks, CTR, and average position.
- Double down on pages with movement; prune weak pages.

## Notes
- The fastest lift usually comes from long-tail intent + internal links, not from repeating a single head keyword.
- If the page is a SPA or PHP template, verify the actual live HTML and not only the source file after deploy.
- Keep the content specific and localized; generic keyword stuffing tends to stall.
