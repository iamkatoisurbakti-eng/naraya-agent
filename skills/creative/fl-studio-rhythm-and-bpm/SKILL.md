# FL Studio Rhythm & BPM

Use this skill when the user asks about rhythm, BPM, beat feel, subdivisions, or how tempo and rhythm work together in FL Studio.

## Core purpose

Rhythm and BPM define how the track moves in time.
Use it to:
- match tempo to musical feel
- place notes on strong and weak beats intentionally
- build rhythmic phrasing
- control bounce, drive, and momentum
- keep the beat aligned across drums, bass, and melody

## Recommended workflow

1. Choose the BPM range first.
2. Decide the rhythmic feel: straight, swing, half-time, double-time, or syncopated.
3. Map the main hits to the grid.
4. Add off-beat notes or syncopation for movement.
5. Check how the rhythm loops over 1, 2, 4, and 8 bars.
6. Adjust note length and spacing to improve pocket.
7. Compare the pattern at the final tempo with metronome or loop.

## Rhythm basics

- Beat: the main pulse
- Subdivision: smaller divisions of the beat
- Meter: how beats group into bars
- Syncopation: accents on unexpected parts of the beat
- Pocket: the comfortable rhythmic feel of the pattern

## BPM and rhythm relationship

- Faster BPM makes rhythmic detail more noticeable.
- Slower BPM gives more space to each hit.
- The same pattern can feel different at different tempos.
- Half-time feel can make fast BPM feel relaxed.
- Double-time feel can make moderate BPM feel energetic.

## Practical rhythm ideas

- Straight rhythm: clean and steady
- Swing rhythm: slightly delayed off-beats for bounce
- Syncopated rhythm: accents between main beats
- Half-time: snare feels slower while tempo stays the same
- Double-time: hats or percussion move faster than the main pulse

## FL Studio workflow tips

- Set BPM before fine rhythm editing.
- Use the grid as a starting point, not a prison.
- Nudge selected notes to improve feel.
- Short notes can make rhythm sharper; longer notes can make it smoother.
- Test with kick and snare first, then add fills and subdivisions.

## Common mistakes

- Choosing BPM without thinking about the feel.
- Overfilling every subdivision.
- Ignoring the relationship between kick, snare, and hat rhythm.
- Making the loop too rigid.
- Letting fast note activity clutter the groove.

## When to use what

- Rhythm: note placement and feel
- BPM: song speed
- Groove: overall pocket
- Swing: shuffled timing feel
- Tempo: pacing of the track
