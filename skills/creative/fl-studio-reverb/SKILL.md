# FL Studio Reverb

Use this skill when the user asks about reverb in FL Studio.

## Core purpose

Reverb adds the sense of space around a sound.
Use it to:
- place sounds in a room or hall
- create depth
- soften a dry element
- add emotion and atmosphere
- blend instruments into a scene

## Recommended workflow

1. Decide the space you want.
2. Choose short, medium, or long decay.
3. Keep the wet level controlled.
4. Filter the reverb if it gets muddy.
5. Use sends when several sounds share the same space.
6. Check that the reverb supports the vocal or lead instead of burying it.

## Practical reverb tips

- Short reverbs feel close and tight.
- Long reverbs feel wide and cinematic.
- Pre-delay can keep the source more present.
- High-pass the reverb return if the low end gets messy.
- Less reverb is often clearer.

## Common mistakes

- Too much reverb everywhere.
- Masking the lead with ambience.
- Using one reverb for every sound without control.
- Letting low frequencies build up in the tail.

## When to use what

- Reverb: space and depth
- Delay: repeats and rhythm
- Mixing: placement in the soundstage
- Arrangement: scene changes and transitions
