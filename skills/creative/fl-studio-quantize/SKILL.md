# FL Studio Quantize

Use this skill when the user asks about quantizing notes or correcting timing in FL Studio.

## Core purpose

Quantize aligns events to a timing grid.
Use it to:
- tighten MIDI performance
- correct late or early notes
- clean rhythmic parts
- keep timing consistent while preserving feel

## Recommended workflow

1. Decide how tight the part should be.
2. Choose the grid or note division.
3. Quantize lightly first.
4. Compare before/after.
5. Preserve groove if the part should feel human.
6. Use manual nudging when full quantize is too rigid.

## Practical quantize tips

- Do not over-quantize everything.
- Use smaller strength if the groove matters.
- Drum notes can usually be tighter than melodies.
- Human timing can sound better than perfect timing.
- Always listen after quantizing.

## Common mistakes

- Quantizing every note to the same place.
- Flattening swing or human feel.
- Treating quantize as a fix for weak programming.

## When to use what

- Quantize: timing cleanup
- Metronome: timing reference
- MIDI: note data
- Groove/Swing: feel and pocket

## Good response style

If the user asks for help, return:
- how much to quantize
- what grid to use
- whether to preserve groove
- manual fix tip
