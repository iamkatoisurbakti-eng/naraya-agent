# Music Generate

## Overview

Use this skill when you need to turn text ideas into music-generation prompts or guide AI music creation.
It helps structure mood, genre, instrumentation, tempo, and song form.

## When to Use

Use when you need to:

- write prompts for AI music generation
- define genre, mood, and instrumentation
- outline song structure
- refine music ideas before generation

Do not use for:

- audio engineering and mixing details
- lyric proofreading only
- tasks that already have a finished composition brief

## Prompt Checklist

- Genre/style
- Mood/emotion
- Tempo/BPM
- Instruments/sound design
- Vocal style or instrumental only
- Structure (intro, verse, chorus, bridge, outro)
- Reference vibe if needed

## Usage Notes

- Keep prompts specific but not overloaded.
- Mention what to avoid as well as what to include.
- Separate vocal instructions from instrumental instructions.
- Use consistent terminology across generations.

## Common Pitfalls

1. Being too vague about genre and mood.
2. Mixing too many styles in one prompt.
3. Forgetting song structure.
4. Overloading the prompt with conflicting references.

## Verification Checklist

- [ ] Genre defined
- [ ] Mood defined
- [ ] Tempo or BPM defined
- [ ] Instrumentation defined
- [ ] Structure defined
