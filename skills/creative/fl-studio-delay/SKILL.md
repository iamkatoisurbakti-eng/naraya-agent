# FL Studio Delay

Use this skill when the user asks about delay in FL Studio.

## Core purpose

Delay repeats a sound over time.
Use it to:
- create rhythmic echoes
- add depth without washing out the source
- make vocal throws or lead accents
- widen or thicken a sound
- support transitions and atmosphere

## Recommended workflow

1. Choose the repeat style.
2. Set tempo-sync if rhythmic repeats matter.
3. Control feedback so the repeats do not overwhelm the mix.
4. Filter the delay return if needed.
5. Use automation for delay throws and transitions.
6. Check whether the repeats add energy or clutter.

## Practical delay tips

- Short delays can thicken without sounding obvious.
- Long delays can create space and drama.
- Ping-pong delay adds stereo movement.
- Filtered delay often sits better in the mix.
- Delay can replace or complement reverb.

## Common mistakes

- Too many repeats.
- Clashing with the main rhythm.
- Delay masking the vocal or lead.
- Using delay without filtering.

## When to use what

- Delay: echoes and rhythmic space
- Reverb: ambient space
- Automation: creative delay throws
- Mixing: clarity and placement
