# FL Studio Mastering Chain

Use this skill when the user asks about a mastering chain in FL Studio.

## Core purpose

A mastering chain is the final processing sequence on a finished mix.
Use it to:
- polish tone
- control peaks
- add subtle cohesion
- prepare the track for export
- keep the master translation safe

## Recommended workflow

1. Start with a clean mix and headroom.
2. Use minimal EQ only if needed.
3. Add gentle compression only if it improves cohesion.
4. Add subtle saturation only if it adds useful color.
5. Put the limiter last.
6. Compare against a reference track.
7. Check the result on multiple systems.

## Common mistakes

- Building the master chain before the mix is strong.
- Too many processors.
- Over-limiting.
- Fixing arrangement issues on the master.

## When to use what

- Mastering chain: final processing order
- Limiter: peak control
- Mixing: balance and cleanup
- Gain: headroom management
