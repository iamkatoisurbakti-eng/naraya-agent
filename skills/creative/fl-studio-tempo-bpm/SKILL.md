# FL Studio Tempo / BPM

Use this skill when the user asks about BPM, tempo, song speed, or how to choose the right pace for a track in FL Studio.

## Core purpose

Tempo sets the energy and pacing of the song.
Use it to:
- match genre expectations
- control groove and urgency
- coordinate drums, melody, and arrangement length
- make the track feel relaxed, steady, or intense

## Recommended workflow

1. Decide the vibe first.
2. Choose a BPM range that fits the genre.
3. Test the loop at that tempo.
4. Adjust groove by nudging notes or changing swing.
5. Check whether the melody and drums still breathe at that speed.
6. Build the arrangement after the tempo feels right.

## Common BPM ranges

These are rough starting points:
- Hip-hop / trap: 70–160 depending on half-time feel
- House: 120–128
- Techno: 125–140
- Pop: 90–130
- R&B: 60–100
- Drum & bass: 160–180
- Lo-fi: 70–95
- Cinematic / ambient: often slower or flexible

## Tempo tips

- BPM changes perceived energy even if the notes stay the same.
- Half-time and double-time feel can make the same BPM work across genres.
- Faster BPM can feel empty if the arrangement is too sparse.
- Slower BPM can still feel intense with strong rhythm and sound design.
- Tempo should support the vocal/melodic phrasing if there is one.

## FL Studio workflow tips

- Set the tempo early so patterns fit correctly.
- If using samples, make sure they sync to the project BPM.
- Try a few BPM values before locking in the final one.
- If the groove feels stiff, test a slightly slower or faster BPM.
- Use time-stretching carefully if audio clips need to follow the project.

## Common mistakes

- Picking BPM too late in the process.
- Choosing a speed that fights the genre.
- Ignoring half-time/double-time perception.
- Too many notes for the chosen pace.
- Not checking how fills and transitions feel at the final tempo.

## When to use what

- Tempo/BPM: song speed and energy
- Scale & key: harmonic center
- Chord progression: harmony movement
- Piano Roll: note detail
- Playlist: arrangement
- Mixer: tone and balance
