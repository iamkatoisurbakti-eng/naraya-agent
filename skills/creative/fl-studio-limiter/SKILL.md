# FL Studio Limiter

Use this skill when the user asks about limiter use in FL Studio.

## Core purpose

A limiter catches peaks and protects the output.
Use it to:
- increase perceived loudness carefully
- prevent clipping
- control final spikes on the master or bus
- finish a mix or master safely

## Recommended workflow

1. Start with a strong mix and headroom.
2. Place the limiter near the end of the chain.
3. Raise level only until the track stays controlled.
4. Watch for pumping, distortion, or flattening.
5. Compare bypass at matched loudness.
6. Stop before the limiter changes the song too much.

## Common mistakes

- Using the limiter to fix a bad mix.
- Pushing too hard and crushing dynamics.
- Putting the limiter too early in the chain.
- Ignoring intersample peaks or hidden distortion.

## When to use what

- Limiter: final peak control
- Compression: dynamic shaping
- Gain: headroom management
- Mastering: release-ready polish

## Good response style

If the user asks for help, return:
- where to place the limiter
- how hard to push it
- warning about pumping/clipping
- next FL Studio step
