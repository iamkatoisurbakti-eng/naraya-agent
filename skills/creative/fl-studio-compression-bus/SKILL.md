# FL Studio Bus Compression

Use this skill when the user asks about compressing a bus or group in FL Studio.

## Core purpose

Bus compression processes a group of related tracks together.
Use it to:
- glue drums, vocals, or music layers
- tame peaks on a group
- create cohesion across related sounds
- add gentle punch or density

## Recommended workflow

1. Route related tracks to the same bus.
2. Decide if you want glue or control.
3. Set gentle compression first.
4. Compare with bypass at matched loudness.
5. Keep the bus from overworking the mix.

## Common uses

- Drum bus glue
- Vocal bus cohesion
- Instrument bus control
- Parallel bus density

## Common mistakes

- Too much gain reduction on a bus.
- Compressing a bus before the individual tracks are balanced.
- Using bus compression to fix arrangement problems.

## When to use what

- Bus compression: group glue/control
- Single-track compression: local dynamics
- Mixing: overall balance
- Gain: level staging
