# FL Studio Playlist Arrangement

Use this skill when the user asks about FL Studio Playlist, arrangement, song structure, automation, transitions, or turning loops into full tracks.

## Core purpose

Playlist is where short patterns become a full song.
Use it to:
- arrange sections
- place patterns on the timeline
- build intros, drops, verses, choruses, bridges, and outros
- add automation clips
- create transitions and tension

## Recommended workflow

1. Finish strong patterns in Channel Rack / Piano Roll first.
2. Place the main loop on the Playlist.
3. Duplicate and vary sections:
   - intro
   - main part
   - breakdown
   - drop/chorus
   - outro
4. Remove or thin elements to create contrast.
5. Add fills and transition effects between sections.
6. Place automation clips for filter, volume, reverb, delay, and build-ups.
7. Check the full timeline for energy flow.
8. Export only after the arrangement feels complete.

## Arrangement principles

- Contrast creates momentum.
- Repetition makes the hook memorable.
- Small changes every 4, 8, or 16 bars help the song feel alive.
- Let sections breathe; do not overcrowd every bar.
- Build energy before a drop, then open the mix after the drop.
- Use silence or reduced instrumentation as a transition tool.

## Playlist tips

- Use color coding for sections if helpful.
- Group similar clips together for readability.
- Keep drum, bass, melody, and FX lanes organized.
- Use pattern blocks for musical parts and audio clips for texture or FX.
- Put automation clips on separate lanes for clarity.
- Zoom out often to check overall song structure.

## Common mistakes

- Making a loop but never arranging a song.
- Copy-pasting the same 8 bars without variation.
- Too many elements fighting at the same time.
- No build-up or transition before major section changes.
- Forgetting automation and dynamics.

## Good section pattern

A common structure:
- Intro
- Verse / Groove
- Build
- Drop / Chorus
- Breakdown
- Final drop / Chorus
- Outro

## When to use what

- Channel Rack: pattern creation
- Piano Roll: detailed notes and melody
- Playlist: arrangement and song flow
- Mixer: mixing and processing
