# FL Studio Scale & Key

Use this skill when the user asks about musical scale, key, key signature, scale highlighting, or how to keep melodies/chords in key in FL Studio.

## Core purpose

Scale and key help keep a song harmonically coherent.
Use them to:
- pick a tonal center
- keep notes inside a scale
- build melodies that sound intentional
- make chords fit the song
- guide basslines and lead lines

## Recommended workflow

1. Decide the mood first.
2. Pick a key that suits the range and vibe.
3. Choose a scale type:
   - major for bright/positive
   - natural minor for darker/emotional
   - harmonic minor for tension and drama
   - pentatonic for simple, catchy ideas
4. Enable scale highlighting if useful.
5. Write the melody around stable scale tones.
6. Build chords from the same key.
7. Check bass notes against the root movement.
8. Test the loop for whether the key feels strong and resolved.

## Practical theory basics

- The key is the tonal home of the song.
- The scale is the set of notes commonly used in that key.
- The tonic/root is the main resting note.
- Chords and melody should usually support the same tonal center.
- A song can borrow notes outside the scale for tension, color, or surprise.

## Scale choices

- Major: upbeat, open, bright
- Natural minor: moody, sad, cinematic
- Harmonic minor: dramatic, tense, exotic feel
- Minor pentatonic: simple, modern, vocal-friendly
- Major pentatonic: clean, uplifting, easy to sing

## Melody tips in key

- Start and end phrases on stable scale tones.
- Use passing notes between chord tones.
- Repeat a motif and change one note for variation.
- If a line feels wrong, check the tonic and strong beats.
- A melody can use non-scale notes briefly if it resolves well.

## Chord tips in key

- Start with triads in the chosen scale.
- Use the diatonic chords as the first draft.
- Borrow a chord only when the emotional effect is wanted.
- Keep the bass/root movement clear.
- Inversions can make the progression smoother.

## Bass tips in key

- Root notes usually define the harmonic floor.
- Use the scale to guide fills and slides.
- Keep bass notes aligned with chord roots when building a basic loop.
- If the bass feels off, simplify it and check the key center.

## FL Studio usage tips

- Use scale highlighting to reduce wrong-note mistakes.
- Use the root note as a visual reference.
- When working fast, sketch in one key first and transpose later if needed.
- If you use MIDI packs, verify the key before combining loops.

## Common mistakes

- Choosing a scale before knowing the emotional goal.
- Writing notes that fight the chord progression.
- Overusing outside notes without resolution.
- Ignoring the root note and tonal center.
- Assuming every loop is in key just because it sounds good alone.

## When to use what

- Piano Roll: note writing and editing
- Channel Rack: pattern sketching
- Playlist: arrangement
- Mixer: balance and tone
