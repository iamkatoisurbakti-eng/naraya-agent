# FL Studio Stereo Widening

Use this skill when the user asks about stereo widening or stereo width in FL Studio.

## Core purpose

Stereo widening increases perceived width in the mix.
Use it to:
- spread pads, effects, or background layers
- make the mix feel larger
- separate supporting elements from the center
- add space without crowding the middle

## Recommended workflow

1. Decide what should stay centered.
2. Widen only elements that benefit from it.
3. Keep bass and kick mostly centered.
4. Check mono compatibility.
5. Compare before/after at matched loudness.

## Practical widening tips

- Wide sides can make a mix feel open.
- The center should still stay strong.
- Use widening on layers, not everything.
- Check phase if a widened sound gets weak in mono.

## Common mistakes

- Widening the low end.
- Making the whole mix too wide and hollow.
- Ignoring mono collapse.
- Using width to hide a weak arrangement.

## When to use what

- Stereo widening: width and spread
- Reverb/delay: space and depth
- Mixing: placement and clarity
- Arrangement: layer choice
