# FL Studio Velocity Control

Use this skill when the user asks about note velocity, dynamics, humanizing MIDI, or making patterns feel more expressive in FL Studio.

## Core purpose

Velocity controls how hard a note is hit.
Use it to:
- create dynamic contrast
- make melodies feel human
- add groove to drums and hats
- emphasize accents
- avoid robotic repetition

## Recommended workflow

1. Identify which notes should be strong and which should be soft.
2. Set the main accents first.
3. Reduce repeated-note uniformity.
4. Add subtle variation to hats, percussion, and melody notes.
5. Check how velocity interacts with sound design and mixer processing.
6. Listen in context and refine.

## Velocity principles

- Strong notes create emphasis.
- Softer notes create bounce and breathing room.
- Velocity can change tone, not only loudness, depending on the instrument.
- Small changes often sound more natural than extreme jumps.
- Repeated identical velocity values can sound machine-like.

## Practical uses

### Drums
- Kick: keep stable, maybe small variation if appropriate
- Snare/clap: accent the backbeat
- Hats: vary velocity to create groove
- Percussion: alternate strong/soft hits for motion

### Melody
- Emphasize phrase endings or important target notes
- Soften passing notes
- Use velocity swells for expression

### Bass
- Keep it controlled unless the instrument reacts musically to dynamics
- Use velocity variation only if it improves articulation

## Humanizing tips

- Do not randomize velocity blindly.
- Accent musical phrases, not every other note.
- Match velocity to the emotional shape of the phrase.
- If the sound gets too harsh, reduce velocity before changing the patch.

## Common mistakes

- Every note at the same velocity.
- Too much randomization with no musical intent.
- Ignoring how velocity changes the instrument tone.
- Over-accenting hats until the groove feels busy.
- Using velocity to solve tone problems that belong in sound design.

## When to use what

- Velocity: dynamics and feel
- Groove: rhythmic pocket
- Piano Roll: detailed editing
- Sound design: tone character
- Mixer: final balance and polish
