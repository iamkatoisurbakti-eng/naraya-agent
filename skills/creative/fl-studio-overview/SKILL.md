# FL Studio Overview

Use this skill when the user says only "FL Studio", "flstudio", or asks for a broad FL Studio workflow without specifying a module.

## Covered areas

- Channel Rack / Step Sequencer
- Playlist arrangement
- Piano Roll note editing
- Mixer routing and mixing
- Scale & key
- Chord progression
- Timing, metronome, count-in, quantize, and MIDI basics
- Drum programming, groove, swing, sidechain, and bus routing
- EQ, compression, gain, reverb, delay, saturation, limiter, and mastering basics
- Arrangement, genre matching, stem handling, and ear training as higher-level production skills

See `references/fl-studio-skill-map.md` for the class-level routing guide and skill family map.

## Quick workflow map

1. Pick key and scale.
2. Build chord progression.
3. Sketch drums and pattern ideas in Channel Rack / Step Sequencer.
4. Write melody, bassline, and detailed notes in Piano Roll.
5. Arrange the full song in Playlist.
6. Route tracks and mix in Mixer.

## How to respond

If the user is vague, ask which area they want:
- beat making / Channel Rack
- arrangement / Playlist
- melody / Piano Roll
- mixing / Mixer
- theory / scale & key
- harmony / chord progression

If the user wants a full workflow, give:
- key/scale suggestion
- chord progression
- drum pattern idea
- melody idea
- arrangement outline
- mix tips

## Common best practices

- Start simple and expand.
- Keep patterns organized.
- Use Piano Roll for detail, Playlist for structure, Mixer for balance.
- Test the song at low volume before finalizing.
- Save variations so you can compare ideas.
