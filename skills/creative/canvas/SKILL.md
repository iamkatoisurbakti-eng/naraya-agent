# Canvas

## Overview

Use this skill when a task involves a canvas-style workspace for arranging, iterating, or composing visual ideas.
It is useful for UI layout exploration, composition, and visual planning.

## When to Use

Use when you need to:

- work on a visual canvas or composition surface
- explore layout alternatives
- arrange elements in a shared workspace
- iterate on visual structure before final output

Do not use for:

- pure text tasks
- backend-only work
- tasks that already have a fixed final layout

## Usage Notes

- Start with rough composition.
- Keep spacing and hierarchy visible.
- Iterate from broad structure to detail.
- Use the canvas to compare alternatives side by side.

## Common Pitfalls

1. Making the layout too dense too early.
2. Ignoring hierarchy and spacing.
3. Treating canvas work as final without iteration.

## Verification Checklist

- [ ] Canvas purpose is clear
- [ ] Layout or composition is explored
- [ ] Alternatives are visible
- [ ] Final structure is ready
