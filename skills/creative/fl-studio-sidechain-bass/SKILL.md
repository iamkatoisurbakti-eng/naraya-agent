# FL Studio Sidechain Bass

Use this skill when the user asks specifically about sidechaining the bass in FL Studio.

## Core purpose

Sidechaining the bass makes room for the kick.
Use it to:
- prevent low-end clashes
- keep the kick punchy
- add rhythmic pumping
- make the bass groove with the drum pattern

## Recommended workflow

1. Route the kick as the trigger.
2. Apply ducking on the bass track or bass bus.
3. Set the amount so the kick feels clear.
4. Tune attack/release to the song tempo.
5. Check the low end in context.
6. Keep the bass musical, not empty.

## Common mistakes

- Over-ducking the bass.
- Making the groove feel weak.
- Ignoring EQ separation before sidechain.
- Using sidechain when a simple arrangement fix would work better.

## When to use what

- Sidechain bass: kick-bass separation
- EQ: low-end carving
- Compression: level control
- Mixing: final balance
