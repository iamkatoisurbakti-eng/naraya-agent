# Inclusive Visuals Specialist

## Overview

Use this skill when visuals need to communicate clearly to diverse audiences while respecting accessibility, readability, and cultural inclusion.

## When to Use

Use when you need to:

- design inclusive graphics or UI visuals
- improve accessibility of visual communication
- choose colors, typography, and layouts with broad readability
- avoid exclusionary, biased, or culturally narrow imagery
- create visuals for mixed-language or mixed-literacy audiences

## Working Style

1. Start from the audience and communication goal.
2. Prefer clarity over decoration.
3. Use high-contrast, readable composition.
4. Avoid stereotypes, ambiguous symbols, or exclusionary cues.
5. Verify the visual works for different viewing conditions.

## Inclusive Design Considerations

- Color contrast and color-blind safety
- Text legibility and hierarchy
- Language simplicity and clarity
- Diverse representation and non-stereotyped imagery
- Respectful cultural context
- Mobile-first readability where relevant

## Common Pitfalls

- Using color alone to convey meaning.
- Choosing tiny text or low-contrast elements.
- Relying on culturally specific references without context.
- Overcrowding the composition.
- Making visuals that look good but communicate poorly.

## Verification Checklist

- [ ] Audience is considered
- [ ] Visual hierarchy is clear
- [ ] Contrast and readability are acceptable
- [ ] Imagery is respectful and inclusive
- [ ] Meaning does not depend on color alone
