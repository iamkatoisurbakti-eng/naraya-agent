# FL Studio Groove & Swing

Use this skill when the user asks about groove, swing, pocket, timing feel, or making patterns move naturally in FL Studio.

## Core purpose

Groove is the feel behind the rhythm.
Use it to:
- make beats bounce
- create pocket and movement
- loosen rigid quantization
- support genre identity
- make loops feel alive

## Recommended workflow

1. Start with a clean rhythm grid.
2. Decide how tight or loose the groove should feel.
3. Add swing only if it supports the style.
4. Shift selected notes slightly off-grid when needed.
5. Use velocity and note length together with timing.
6. Compare the loop with and without groove changes.
7. Preserve the core pocket while adding variation.

## Groove principles

- Groove is not just swing; it is the total rhythmic feel.
- Timing, velocity, note length, and silence all affect groove.
- A beat can be tight and still groovy.
- Too much looseness can make the rhythm feel sloppy.
- Genre matters: some styles want precision, others want bounce.

## Practical groove tools

- Swing control
- Slight note offset
- Velocity variation
- Shorter or longer note lengths
- Ghost notes and pickup hits
- Call-and-response rhythmic shapes

## Groove tips by section

- Verse: often more restrained and pocket-focused
- Chorus/drop: often wider, stronger, or more energetic
- Breakdown: groove may open up or simplify
- Fills: brief rhythm disruption before returning to pocket

## Common mistakes

- Quantizing everything to the same rigid grid.
- Using swing without checking the genre.
- Over-shifting notes until the beat drags.
- Ignoring velocity and note length.
- Making every section groove the same way.

## When to use what

- Groove: overall rhythmic feel
- Swing: grid-based shuffle feel
- Velocity: dynamic variation
- Piano Roll: timing/note shaping
- Channel Rack: quick groove sketching
