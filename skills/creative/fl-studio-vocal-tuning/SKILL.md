# FL Studio Vocal Tuning

Use this skill when the user asks about tuning vocals in FL Studio.

## Core purpose

Vocal tuning corrects pitch while keeping the performance musical.
Use it to:
- fix off-pitch notes
- smooth melody movement
- keep harmony parts aligned
- create stylized tuning if desired

## Recommended workflow

1. Listen to the take first.
2. Fix the worst notes before heavy tuning.
3. Tune gently when natural sound matters.
4. Tune harder only if the style calls for it.
5. Recheck timing and phrasing after pitch work.
6. Compare tuned and untuned versions.

## Common mistakes

- Over-tuning a performance that should sound human.
- Fixing pitch without checking timing.
- Tuning the entire part identically.
- Ignoring the emotional shape of the vocal.

## When to use what

- Vocal tuning: pitch correction
- De-esser: sibilance control
- Compression: dynamics
- Vocal processing: full chain
