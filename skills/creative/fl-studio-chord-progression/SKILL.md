# FL Studio Chord Progression

Use this skill when the user asks about chord progression, harmony, chord loops, voice leading, or building chords in FL Studio.

## Core purpose

Chord progression gives the song emotional direction.
Use it to:
- establish mood
- support melody and bass
- create tension and release
- shape section changes
- give the track a memorable harmonic identity

## Recommended workflow

1. Choose the key and scale first.
2. Pick a harmonic mood:
   - bright
   - sad
   - tense
   - dreamy
   - triumphant
3. Start with a simple 2-, 4-, or 8-chord loop.
4. Use diatonic chords as the first draft.
5. Check whether the progression resolves clearly.
6. Adjust voicings so the movement feels smooth.
7. Add inversion, extension, or borrow notes only when needed.
8. Test the chord loop with melody and bass.

## Common progression formulas

These are starting points, not rules:
- I–V–vi–IV
- vi–IV–I–V
- i–VI–III–VII
- i–iv–V–i
- I–vi–IV–V
- ii–V–I for stronger resolution in some styles

## Voice leading

Voice leading is how the notes move from one chord to the next.
Good voice leading:
- reduces awkward jumps
- keeps the harmony smooth
- makes progressions feel intentional

Tips:
- move each note the shortest useful distance
- keep common tones when possible
- use inversions to reduce motion
- avoid unnecessary large jumps in every voice

## Chord choice tips

- Start with triads before adding extensions.
- Seventh chords can add richness.
- Suspended chords can delay resolution.
- Borrowed chords can create surprise or color.
- Too many complex chords too early can muddy the idea.

## Progression design ideas

- Keep verse harmony simpler than chorus harmony.
- Use a turnaround before repeating the loop.
- Build tension in the pre-chorus by changing chord rhythm or direction.
- Let the chorus feel more open or resolved than the verse.
- Repeating the same progression can work if the arrangement changes.

## FL Studio workflow tips

- Build the loop in Piano Roll.
- Test chords with the bass root notes.
- Copy the progression to multiple patterns for different sections.
- Use inversion to keep the chord register clear.
- If the progression feels empty, add a top note melody or pad.

## Common mistakes

- Chords that fight the melody.
- Too much harmonic movement with no resting point.
- Dense voicings in the low register.
- Progressions that loop but never create a section change.
- Using fancy chords before the core loop works.

## When to use what

- Chord progression: harmony and emotional movement
- Scale/key: note family and tonal center
- Piano Roll: note layout and voicings
- Playlist: section placement and variation
- Mixer: tonal balance and glue
