# FL Studio Punch-In

Use this skill when the user asks about punch-in recording or retaking a specific section in FL Studio.

## Core purpose

Punch-in lets you record only part of a performance.
Use it to:
- fix a bad line or phrase
- replace a small timing mistake
- keep the rest of the take intact
- do clean overdubs

## Recommended workflow

1. Locate the problem section.
2. Set a clear punch-in and punch-out point.
3. Rehearse the entrance before recording.
4. Keep the metronome or count-in on.
5. Record only the needed section.
6. Crossfade or smooth the transition if necessary.

## Common mistakes

- Punching in too late.
- Not rehearsing the entrance.
- Leaving an obvious transition between old and new audio.

## When to use what

- Punch-in: partial retake
- Count-in: recording preparation
- Editing: fixing timing or transitions
- Vocal processing: polishing the retake
