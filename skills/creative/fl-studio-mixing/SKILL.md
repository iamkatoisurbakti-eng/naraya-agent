# FL Studio Mixing

Use this skill when the user asks about mixing music in FL Studio.

## Core purpose

Mixing balances and polishes the track.
Use it to:
- set relative volume balance
- clean frequencies
- control dynamics
- manage stereo space
- add space and depth with effects
- prepare the track for export/mastering
- decide when a narrower sub-skill should take over (EQ, compression, reverb, delay, etc.)

## Recommended workflow

1. Set rough balance first.
2. Clean problematic frequencies with EQ.
3. Apply compression where needed.
4. Place sounds in the stereo field.
5. Use reverb and delay with intention.
6. Group and route similar sounds.
7. Check the mix on multiple playback levels.
8. Compare against reference tracks if possible.

## Mixing priorities

- Volume balance
- Low-end control
- Midrange clarity
- High-end smoothness
- Stereo placement
- Depth and space

## Practical mixing tips

- Balance before plugin stacking.
- EQ out mud before boosting brightness.
- Keep bass and kick from fighting.
- Avoid over-widening the low end.
- Use bus processing for related instruments.
- Check the mix in mono sometimes.

## Common mistakes

- Mixing too loudly.
- Too much reverb everywhere.
- No headroom.
- Clashing low frequencies.
- Over-compressing.
- Ignoring reference listening.

## When to use what

- Mixing: overall polish and balance
- Compression: dynamics
- Gain: level staging
- Arrangement: energy over time
- Sound design: source tone

## Good response style

If the user asks for help, return:
- mix priority
- EQ/compression tip
- routing idea
- stereo/depth idea
- next FL Studio step

If the request is narrowly about one processor or technique, route to the specialized skill instead of expanding this one into a flat list.
