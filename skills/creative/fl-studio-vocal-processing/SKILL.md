# FL Studio Vocal Processing

Use this skill when the user asks about vocals, vocal chains, vocal recording, vocal tuning, or vocal mixing in FL Studio.

## Core purpose

Vocal processing makes vocals clearer, more controlled, and more musical.
Use it to:
- clean recorded vocals
- tune and align performance
- reduce noise and harshness
- control dynamics
- add space, vibe, and polish
- fit vocals into the instrumental

## Recommended workflow

1. Choose the best take or comp the vocal.
2. Clean silence, pops, and obvious noise.
3. Tune pitch if needed.
4. Correct timing and phrasing if needed.
5. Apply EQ to remove mud and harsh buildup.
6. Compress to control dynamics.
7. Add de-essing if sibilance is strong.
8. Add reverb/delay with intention.
9. Match vocal level to the beat.
10. Recheck in the full mix.

## Common vocal chain ideas

- Cleanup / noise reduction
- EQ
- Compression
- De-esser
- Saturation or mild enhancement
- Reverb / delay sends
- Automation for level and emphasis

## Practical vocal tips

- Record cleanly before processing.
- Avoid over-tuning unless that is the style.
- Keep the vocal forward without making it harsh.
- Use sends for reverb/delay when possible.
- Automate phrases so the vocal stays intelligible.

## Common mistakes

- Too much reverb causing distant vocals.
- Over-compression making the vocal lifeless.
- Ignoring sibilance.
- Adding effects before cleaning the take.
- Fitting the vocal to the instrumental instead of balancing both.

## When to use what

- Vocal processing: complete vocal chain
- Compression: dynamic control
- Gain: input/output level management
- Mixing: placement in the track
- Ear training: judging vocal tone and balance
