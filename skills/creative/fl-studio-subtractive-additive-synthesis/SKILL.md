# FL Studio Subtractive / Additive Synthesis

Use this skill when the user asks about subtractive synthesis, additive synthesis, synth patch design, or how to build tones in FL Studio.

## Core purpose

These synthesis methods are two common ways to create and shape sound.
Use them to:
- design custom timbres
- shape brightness and tone
- build leads, basses, pads, and textures
- understand how harmonics affect sound

## Subtractive synthesis

Subtractive synthesis starts with a harmonically rich sound and removes parts of it.

Typical flow:
1. Start with a saw, square, or rich oscillator/sample.
2. Use a filter to cut unwanted frequencies.
3. Shape amplitude with envelopes.
4. Add movement with LFO or modulation.
5. Polish with effects like saturation, chorus, or reverb.

Why use it:
- simple and flexible
- great for warm, classic synth sounds
- easy to shape into leads, basses, pads, and plucks

## Additive synthesis

Additive synthesis builds a sound by combining harmonics/partials.

Typical flow:
1. Start from simple sine-like components or harmonic layers.
2. Add partials to define tone.
3. Adjust harmonic balance for brightness or color.
4. Use modulation to animate the tone.
5. Refine with effects if needed.

Why use it:
- precise control over harmonic content
- useful for organ-like, glassy, or evolving tones
- good for designing unusual textures and rich timbres

## Harmonic thinking

- Low harmonics make a sound feel warmer or darker.
- More high harmonics make it brighter or sharper.
- A filter removes or reshapes harmonic content.
- Overtones influence the perceived character more than the fundamental alone.

## Practical sound goals

- Bass: controlled low end, limited harsh harmonics
- Lead: clear attack and defined upper harmonics
- Pad: wide, smooth harmonic spread
- Pluck: bright attack with quick decay
- Texture: evolving harmonic movement

## FL Studio workflow tips

- Start simple and add complexity only when needed.
- Listen to the sound in the mix, not only solo.
- Use envelopes, filters, and modulation intentionally.
- If a patch feels harsh, remove harmonics before adding more processing.
- If a patch feels dull, add harmonics or brighten the filter.

## Common mistakes

- Adding effects before the core tone is good.
- Making sounds too wide in the low end.
- Ignoring envelope shape.
- Designing without considering the song context.
- Confusing brightness with loudness.

## When to use what

- Subtractive: rich source -> remove/shape
- Additive: build sound from partials/harmonics
- Piano Roll: note control
- Mixer: polish and balance
- Playlist: arrange the patch in the song
