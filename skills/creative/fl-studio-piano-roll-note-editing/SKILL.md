# FL Studio Piano Roll Note Editing

Use this skill when the user asks about FL Studio Piano Roll, detailed MIDI editing, melody writing, chords, basslines, slides, rolls, or expressive note programming.

## Core purpose

Piano Roll is where patterns become musical and expressive.
Use it to:
- write melodies
- program basslines
- build chord progressions
- create arps, runs, and fills
- edit note length, timing, velocity, and articulation

## Recommended workflow

1. Start from a simple motif or chord idea.
2. Lay down the main notes.
3. Adjust note length and timing for groove.
4. Use velocity to create dynamics.
5. Add variation by moving a few notes off-grid intentionally.
6. Duplicate and modify phrases instead of rewriting from scratch.
7. Add rolls, slides, or ornamentation where the genre needs it.
8. Test the loop with drums and bass before finalizing.

## Piano Roll principles

- Melody should be singable or memorable.
- Bass should support rhythm and harmony.
- Chords should leave room for lead and vocal space.
- Small rhythmic changes often sound better than constant motion.
- Velocity variation prevents MIDI from sounding robotic.
- Note length matters as much as pitch.

## Melody tips

- Build around a strong motif.
- Use repetition with one small change.
- Leave rests; silence helps the melody breathe.
- Keep the contour clear: rise, fall, or answer.
- End phrases on stable notes when you want resolution.
- Use passing notes sparingly to connect targets.

## Bassline tips

- Lock the bass with kick rhythm when needed.
- Keep the low end simple if the arrangement is busy.
- Use shorter notes for punch, longer notes for sustain.
- Let the bass answer the kick instead of competing with it.
- Add slides or glides only when genre-appropriate.

## Chord tips

- Start with triads or simple voicings.
- Invert chords to smooth motion.
- Avoid crowding the low register.
- Spread voicings to keep the mix open.
- Use passing chords or suspensions for movement.

## Common Piano Roll tools

- Draw/paint notes for fast entry
- Select and nudge notes for timing
- Quantize carefully; do not over-quantize everything
- Velocity editor for dynamics
- Slide notes for pitch movement where supported
- Arpeggiation for fast broken-chord ideas
- Chopping/shortening notes for rhythmic variation

## Common mistakes

- Too many notes in the same space.
- No rhythmic variation between phrases.
- Over-quantized, mechanical feel when groove is needed.
- Bass and melody fighting in the same register.
- Chords that are too dense for the arrangement.

## When to use Piano Roll vs other tools

- Piano Roll: detailed pitch/rhythm editing
- Channel Rack: quick pattern sketching
- Playlist: arrangement and structure
- Mixer: tone shaping and mixing
