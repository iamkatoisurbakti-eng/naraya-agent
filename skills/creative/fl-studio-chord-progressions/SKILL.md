# FL Studio Chord Progressions

Use this skill when the user asks for chord progression ideas or quick harmony patterns in FL Studio.

## Core purpose

Chord progressions give the track direction and emotional motion.
Use them to:
- create mood
- support the melody
- add tension and release
- establish verse/chorus contrast
- make the loop feel musical instead of static

## Fast workflow

1. Pick the key and scale.
2. Choose the mood.
3. Start with a simple 4-chord loop.
4. Test the loop with bass and melody.
5. Adjust one chord at a time for smoother motion or stronger emotion.
6. Use inversions if the jump between chords feels too large.

## Common starter ideas

- I–V–vi–IV
- vi–IV–I–V
- i–VI–III–VII
- i–iv–V–i
- I–vi–IV–V

## Practical tips

- Keep verse harmony simpler than chorus harmony.
- Use repeat with variation instead of constant chord changes.
- Add a turnaround to make the loop feel complete.
- Try suspended or extended chords only after the basic loop works.
- Smooth voice leading usually sounds better than big jumps everywhere.

## FL Studio tips

- Build chords in Piano Roll.
- Duplicate the loop into different patterns for song sections.
- Layer a pad or top melody if the progression feels empty.
- Check the bass root notes against the chord roots.

## Common mistakes

- Too many fancy chords too early.
- Chords that fight the melody.
- No tension/release shape.
- Static loop with no section change.
