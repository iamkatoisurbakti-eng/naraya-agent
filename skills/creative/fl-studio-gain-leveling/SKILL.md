# FL Studio Gain & Leveling

Use this skill when the user asks about gain, leveling, headroom, or input/output balance in FL Studio.

## Core purpose

Gain and leveling are the foundation of a clean mix.
Use them to:
- prevent clipping
- create headroom
- balance sources before mixing
- feed plugins at a sensible level
- keep the master clean

## Recommended workflow

1. Set source levels first.
2. Trim loud sounds before adding effects.
3. Keep headroom on individual tracks and the master.
4. Match loudness when comparing changes.
5. Use faders and gain controls for balance, not only plugins.
6. Re-check levels after processing.

## Practical gain tips

- Gain staging starts before compression and EQ.
- If a plugin sounds bad, check the level going into it.
- Lower source gain instead of fighting clipping downstream.
- Use headroom so the mix has room to breathe.
- Keep sub/low-end sources especially controlled.

## Common mistakes

- Running everything too hot.
- Fixing level issues only at the master.
- Adding effects before controlling input level.
- Confusing loudness with quality.
- Ignoring how gain affects compressor behavior.

## When to use what

- Gain: input/output level control
- Compression: dynamic shaping
- Mixing: balance and clarity
- Arrangement: relative energy across sections
