# FL Studio MIDI

Use this skill when the user asks about MIDI in FL Studio.

## Core purpose

MIDI is performance data, not audio.
Use it to:
- record notes from a keyboard or controller
- edit melodies, chords, and drums
- trigger virtual instruments
- change note length, velocity, and timing
- route control data to instruments and plugins

## Recommended workflow

1. Connect the MIDI device or create notes in Piano Roll.
2. Record or draw the performance.
3. Edit timing, pitch, and velocity.
4. Assign the right instrument.
5. Test the part in context.
6. Convert to audio only when needed.

## Practical MIDI tips

- MIDI is flexible, so keep it editable.
- Velocity can shape feel before mixing.
- Split or layer MIDI parts for better arrangement.
- Use MIDI for composition and audio for final texture.

## Common mistakes

- Treating MIDI like finished audio.
- Ignoring velocity and note length.
- Not checking the instrument mapping.

## When to use what

- MIDI: note performance data
- Piano Roll: note editing
- Quantize: timing cleanup
- Velocity: expression

If the user asks for help, return:
