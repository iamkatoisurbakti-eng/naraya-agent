# FL Studio Count-In

Use this skill when the user asks about count-in or pre-roll before recording in FL Studio.

## Core purpose

Count-in gives a clear lead-in before recording starts.
Use it to:
- prepare the performer before the take
- avoid missed entrances
- start on the bar cleanly
- support timing when recording vocals or instruments

## Recommended workflow

1. Set the BPM.
2. Enable the metronome.
3. Turn on count-in or pre-roll.
4. Rehearse the entrance once.
5. Record on the click after the count.
6. Review whether the start lands cleanly.

## Common mistakes

- Recording without any lead-in.
- Starting too late after the count.
- Using count-in but ignoring the actual tempo feel.

## When to use what

- Count-in: pre-record entry
- Metronome: timing reference
- Quantize: note timing cleanup
- MIDI: recorded performance data

## Good response style

If the user asks for help, return:
- whether to use count-in
- how many bars to count
- timing advice
- next FL Studio step
