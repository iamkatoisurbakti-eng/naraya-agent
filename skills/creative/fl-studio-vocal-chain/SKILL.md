# FL Studio Vocal Chain

Use this skill when the user asks about a vocal chain in FL Studio.

## Core purpose

A vocal chain organizes the processing order for lead vocals.
Use it to:
- clean the vocal
- correct pitch
- control sibilance
- manage dynamics
- add space and polish

## Recommended workflow

1. Clean the recording.
2. Tune pitch if needed.
3. Use EQ to remove problem frequencies.
4. De-ess harsh sibilance.
5. Compress for stable level.
6. Add color or saturation lightly if needed.
7. Send to reverb and delay.
8. Rebalance in the full mix.

## Common mistakes

- Too many processors too early.
- Reverb and delay masking the lyric.
- Compression before cleanup when the vocal is noisy.
- Ignoring the song style when choosing the chain.

## When to use what

- Vocal chain: full lead vocal processing order
- Vocal tuning: pitch correction
- De-esser: sibilance control
- Compression: dynamic control
- Reverb/delay: space and vibe
