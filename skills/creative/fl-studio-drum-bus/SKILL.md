# FL Studio Drum Bus

Use this skill when the user asks about a drum bus in FL Studio.

## Core purpose

A drum bus groups drum elements together.
Use it to:
- glue kick, snare, hats, and percussion
- add punch or density
- process drums as one unit
- keep the drum mix coherent

## Recommended workflow

1. Route all drum elements to the drum bus.
2. Balance individual drum tracks first.
3. Add gentle bus compression if needed.
4. Add saturation or EQ only if it improves the group.
5. Compare before/after at matched loudness.
6. Keep the bus processing subtle.

## Common mistakes

- Overcompressing the drum bus.
- Using the bus to fix bad drum programming.
- Destroying transients.
- Making the drum bus too loud and harsh.

## When to use what

- Drum bus: group processing
- Compression: glue and control
- Saturation: density and color
- Mixing: individual balance first

## Good response style

If the user asks for help, return:
- which drum group to route
- gentle processing order
- punch vs glue note
- next FL Studio step
