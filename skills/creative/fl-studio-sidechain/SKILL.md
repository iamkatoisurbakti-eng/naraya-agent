# FL Studio Sidechain

Use this skill when the user asks about sidechain, ducking, or kick-bass interaction in FL Studio.

## Core purpose

Sidechain makes one sound react to another.
Use it to:
- create kick and bass separation
- add pumping movement
- clear space for a lead or vocal
- make dynamics breathe in time with the rhythm

## Recommended workflow

1. Decide what should trigger the ducking.
2. Decide what should be reduced.
3. Route the sidechain signal.
4. Set the amount and timing of ducking.
5. Listen in the full mix.
6. Adjust until it feels musical, not overdone.

## Common uses

- Kick ducks bass
- Vocal ducks pads or music bed
- Snare-triggered movement for effects
- Pumping rhythm in dance music

## Common mistakes

- Ducking too hard.
- Using sidechain when simple EQ is enough.
- Making the groove disappear.
- Forgetting the sidechain should serve the song.

## When to use what

- Sidechain: dynamic interaction
- Compression: level control
- EQ: frequency separation
- Mixing: space and balance

If the user asks for help, return:
