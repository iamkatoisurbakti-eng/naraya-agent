# FL Studio Master Bus Chain

Use this skill when the user asks about a master bus chain in FL Studio.

## Core purpose

A master bus chain is the final processing path on the master output.
Use it to:
- apply subtle polish
- manage final tone and peaks
- keep the mix release-ready
- avoid overprocessing the master

## Recommended workflow

1. Start with a strong mix and headroom.
2. Keep the chain simple.
3. Use light EQ only if needed.
4. Add gentle compression only if it improves cohesion.
5. Control peaks with limiting at the end.
6. Compare against a reference track.
7. Check translation on different speakers or headphones.

## Common chain idea

- Subtle EQ
- Gentle compression
- Optional saturation
- Limiter
- Output check

## Common mistakes

- Fixing mix problems on the master.
- Too many processors.
- Over-limiting and flattening the song.
- Ignoring the mix stage before mastering.

## When to use what

- Master bus chain: final polish path
- Mixing: balance and cleanup
- Mastering: final loudness and translation
- Gain: headroom
