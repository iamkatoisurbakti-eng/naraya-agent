# FL Studio Metronome

Use this skill when the user asks about the metronome, timing, count-in, or staying in tempo in FL Studio.

## Core purpose

The metronome is the timing reference.
Use it to:
- keep parts locked to tempo
- practice rhythm and timing
- record with a count-in
- check whether a pattern drifts
- build tighter performances

## Recommended workflow

1. Set the BPM first.
2. Turn on the metronome.
3. Use count-in before recording if needed.
4. Record or program to the click.
5. Listen for rushing or dragging.
6. Adjust placement or timing if the performance feels off.
7. Turn the metronome off when evaluating musical feel, if needed.

## Practical metronome tips

- Use a simple click to train timing.
- Practice clapping or tapping with the click.
- Start slow if the rhythm is difficult.
- For recording, a count-in helps avoid late starts.
- If the groove feels stiff, compare click timing with swing or humanized placement.

## Common mistakes

- Setting tempo after recording instead of before.
- Ignoring count-in when recording.
- Practicing only at fast tempos.
- Confusing tight timing with robotic timing.

## When to use what

- Metronome: timing reference and practice
- Tempo/BPM: song speed
- Groove/Swing: feel and pocket
- Arrangement: song flow
- Ear training: judging timing accuracy
