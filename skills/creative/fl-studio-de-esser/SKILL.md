# FL Studio De-Esser

Use this skill when the user asks about de-essing or reducing harsh vocal sibilance in FL Studio.

## Core purpose

A de-esser reduces sharp consonant energy, especially s, sh, and t sounds.
Use it to:
- smooth harsh vocals
- control piercing high-frequency spikes
- keep the vocal present without sounding abrasive
- make vocal compression more usable

## Recommended workflow

1. Find the harsh range by ear.
2. Set de-essing only on the problem area.
3. Reduce just enough to soften the sibilance.
4. Compare bypass at matched loudness.
5. Re-check after compression and EQ.

## Common mistakes

- Over-de-essing until the vocal lisps.
- Using it when the real problem is bad recording or EQ.
- Targeting the wrong frequency area.

## When to use what

- De-esser: sibilance control
- EQ: tonal shaping
- Compression: overall dynamics
- Vocal tuning: pitch correction

## Good response style

If the user asks for help, return:
- harshness area to target
- mild or stronger reduction
- placement in the chain
- cleanup note
