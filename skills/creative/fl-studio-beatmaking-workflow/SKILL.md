# FL Studio Beatmaking Workflow

Use this skill when the user asks for the full beatmaking process in FL Studio from start to finish.

## Core purpose

This skill ties the whole FL Studio process together.
Use it to:
- turn an idea into a beat
- organize the production stages
- avoid getting stuck in one window
- move from sketch to arrangement to mix
- keep the workflow fast and practical
- route the user into the right sub-skill when they ask for a narrower task

For the class-level map, see `references/fl-studio-skill-map.md`.

## Recommended end-to-end flow

1. Choose genre, mood, key, and BPM.
2. Build the drum backbone in Channel Rack / Step Sequencer.
3. Write melody, bass, and chords in Piano Roll.
4. Shape groove with velocity, swing, and timing.
5. Create sound design layers if needed.
6. Arrange the song in Playlist.
7. Route tracks into the Mixer.
8. Balance, EQ, compress, and add send effects.
9. Automate transitions and energy changes.
10. Export and review.

## Fast workflow principles

- Start with the most important idea first.
- Do not perfect one loop forever.
- Move between tools as the task changes.
- Use patterns for idea generation, Playlist for structure, Mixer for polish.
- Keep the production moving from rough to refined.

## Typical beatmaking order

### 1. Foundation
- BPM
- key/scale
- drum pocket
- main harmonic idea

### 2. Musical core
- chord progression
- melody hook
- bassline
- groove adjustments

### 3. Sound and texture
- synth design
- counter-melodies
- effects
- ambience

### 4. Arrangement
- intro
- main section
- variation
- drop/chorus
- breakdown
- outro

### 5. Mix
- routing
- leveling
- EQ
- compression
- stereo space
- sends

## Common mistakes

- Starting arrangement before the loop works.
- Mixing too early before the musical idea is clear.
- Using too many plugins before the basics are solid.
- Repeating the same loop without section changes.
- Ignoring groove, velocity, and sound choice.
