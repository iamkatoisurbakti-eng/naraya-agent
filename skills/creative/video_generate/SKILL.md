# Video Generate

## Overview

Use this skill when you need to turn text ideas into AI video prompts or structure a video concept for generation.
It helps define scenes, motion, pacing, camera behavior, and output format.

## When to Use

Use when you need to:

- write prompts for AI video generation
- define scene flow and transitions
- specify motion, pacing, and camera direction
- refine a video concept before rendering

Do not use for:

- static image prompts only
- editing a finished video file
- audio-only production tasks

## Prompt Checklist

- Subject and scene
- Motion and action
- Camera behavior
- Style and mood
- Duration or pacing
- Aspect ratio / output format
- What to avoid

## Usage Notes

- Keep the shot list clear and short.
- Mention camera movement only when it matters.
- State the desired pacing explicitly.
- Separate visual direction from audio direction.

## Common Pitfalls

1. Leaving scene motion undefined.
2. Mixing too many visual styles.
3. Forgetting the output format or aspect ratio.
4. Writing prompts that are too long and conflicting.

## Verification Checklist

- [ ] Scene defined
- [ ] Motion defined
- [ ] Camera behavior defined
- [ ] Pacing defined
- [ ] Output format defined
